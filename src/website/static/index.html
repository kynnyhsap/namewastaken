[ Usage ]

Get started in seconds.

No installation required. Just run with npx.

CLI
# Check a username npx namewastaken mrbeast # Check specific platforms npx nwt mrbeast -p tt,ig,yt # JSON output npx nwt mrbeast --json
SDK
import nwt from 'namewastaken' const result = await nwt.check('mrbeast') result.tiktok.taken // true await nwt.available('mrbeast') // false
